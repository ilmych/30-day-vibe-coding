## The Workflow

I build with Claude Code using a conversational approach:

1. **I describe** what I want in plain language
2. **Claude proposes** a design and architecture
3. **I approve** or suggest changes
4. **Claude builds** the entire app
5. **I test** on device/browser
6. **I report** bugs in plain language ("tapping doesn't work")
7. **Claude fixes** based on my reports
8. **Claude publishes** to GitHub/GitHub Pages

My actual typing per app: ~5-10 short messages. Claude writes thousands of lines of code.
